# Graphic Designer Landing Page

A modern, responsive landing page designed specifically for a graphic designer to showcase their portfolio, services, and skills.

## Features

- **Responsive Design**: Works on all device sizes from mobile to desktop
- **Portfolio Showcase**: Grid layout to display design projects with hover effects
- **Services Section**: Highlights key design services offered
- **About Section**: Professional bio and skill display
- **Contact Form**: Functional contact form with validation
- **Smooth Navigation**: Smooth scrolling and fixed navigation
- **Modern UI**: Clean, professional design with animations

## Files Included

- `index.html`: Main HTML structure
- `styles.css`: All styling and responsive design
- `script.js`: Interactive functionality and animations

## How to Use

1. Open `index.html` in a web browser to view the landing page
2. To customize for your own use:
   - Update the name and contact information in `index.html`
   - Replace placeholder images with your own portfolio
   - Modify colors in `styles.css` to match your brand
   - Update the content in each section to reflect your services and bio

## Customization

### Colors
- Primary color: `#4a6cf7` (blue)
- You can change this by updating the color values in `styles.css`

### Fonts
- Uses Google Fonts Poppins family
- Easily change by updating the font link in `index.html` and CSS

### Images
- All images are from Unsplash
- Replace with your own by updating the `src` attributes in `index.html`
